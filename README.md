## Purpose 用途

用于教师及同学在校园内给学校设施提交在线报修申请。

## Usage 用途

:one: 下载所有文件，并上传至您的服务器（假设放在 /RMS/ 目录下）

:two: 将文件包内的sample.sql导入到数据库中

:three: 打开Functions/PDOConn.php，修改数据库连接配置

:four: 访问example.com/RMS，即可使用初始密码(下附)登录。
